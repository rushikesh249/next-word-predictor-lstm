## Screenshots Guidelines

When adding screenshots:

1. **Frontend Screenshots**: Capture the web interface showing:
   - Connection status (connected/disconnected)
   - Text input with example text
   - Prediction results with word breakdown
   - Mobile responsive view

2. **Backend Screenshots**: Show terminal output with:
   - Server startup messages
   - Model loading confirmation
   - API request/response logs
   - Training progress (if applicable)

3. **Performance Charts**: Include:
   - Sequential vs OpenMP timing comparison
   - Speedup metrics visualization
   - Dataset size vs processing time graphs
