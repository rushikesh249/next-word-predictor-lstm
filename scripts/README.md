# Scripts

Utility scripts for project management and analysis.

## Files

- `start_project.py` - Automated project startup script
- `test_integration.py` - Integration testing script
- `graph.py` - Performance visualization script

## Usage

### Start Project (Recommended)
```bash
python scripts/start_project.py
```
This script will:
- Check and install dependencies
- Start the backend server
- Open the frontend in browser
- Provide status updates

### Test Integration
```bash
python scripts/test_integration.py
```
Tests the complete system integration.

### Generate Performance Graph
```bash
python scripts/graph.py
```
Creates performance comparison charts.
